表单

Learn from this

-- 01 -- 
      Template Driven Forms - 模板驱动式表单
        - 使用方便
        - 适用于简单的场景
        - 通过 [(ngModel)] 实现数据双向绑定
        - 自动生成 Form Model (异步)
        - 最小化组件类的代码
        - 不易于单元测试
-- 02 -- 
      Reactive Forms - 响应式表单
        - 比较灵活
        - 适用于复杂的场景
        - 简化了HTML模板的代码，把验证逻辑抽离到组件类中
        - 手动创建 Form Model (同步)
        - 方便的跟踪表单控件值的变化
        - 易于动态添加表单控件
        - 易于单元测试
